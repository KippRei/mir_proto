audio_preprocessor.py
 
The AudioPreprocessor class handles the preprocessing of audio files. The files
are separated into stems (drums, bass, melody, vocals), the tempo is analyzed and adjusted
to a set BPM (126), and then converted and saved as NumPy arrays
for the audio engine.
 
Preprocessing is executed in a separate thread to avoid blocking the program.
